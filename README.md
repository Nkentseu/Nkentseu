# Nkentseu
Graphics Engine

## Qu'est-ce que Nkentseu Engine ?
Nkentseu Engine est un moteur graphique haute performance en cours de développement par TEUGUIA TADJUIDJE Rodolf Séderis étudiant en 5éme année à l'école national supérieur polytechnique de Yaoundé au Cameroun.
Le moteur graphique doit être capable plus tard de développer d'autre application tel que :

- Nkentseu Game Engine : Moteur de jeu vidéo baser sur le noyau Nkentseu
- Nkentseu Simulator : Une Application de simulation 3D
- etc...

## Avancement actuel
Pour l'instant le moteur va tourner avec :

- SDL : API de fenêtrage par défaut qui va aussi nous permettre d'avoir une application multi plateforme plus tard
- Glad : une implémentation de OpenGL modern pour les appareils qui pourront supporter OpenGL
- Vulkan : pour les applications qui vont supporter Vulkan

Il n'y aura pas pour l'instant une application pour DirectX bien que SDL2 supporte DirectX 12
